# Access Geographic Data

Functions for downloading of geographic data for use in spatial analysis
and mapping. The package facilitates access to climate, crops,
elevation, land use, soil, species occurrence, accessibility,
administrative boundaries and other data.

|                                                                                              |                                                                                 |
|----------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------|
| **Function**                                                                                 | **Description**                                                                 |
| [`bio_oracle`](https://rspatial.github.io/geodata/reference/biooracle.md)                    | Marine data from bio-oracle                                                     |
| [`cmip6_world`](https://rspatial.github.io/geodata/reference/cmip6.md)                       | Downscaled and calibrated CMIP6 projected future climate data                   |
| [`cmip6_tile`](https://rspatial.github.io/geodata/reference/cmip6.md)                        | Downscaled and calibrated CMIP6 data by tile                                    |
| [`country_codes`](https://rspatial.github.io/geodata/reference/countries.md)                 | Country codes                                                                   |
| [`crop_calendar_sacks`](https://rspatial.github.io/geodata/reference/crop_calendar_Sacks.md) | Sachs crop calendar data                                                        |
| [`crop_monfreda`](https://rspatial.github.io/geodata/reference/crop_monfreda.md)             | Monfreda crop data (area, yield)                                                |
| [`crop_spam`](https://rspatial.github.io/geodata/reference/crop_spam.md)                     | SPAM crop data (area, yield, value)                                             |
| [`cropland`](https://rspatial.github.io/geodata/reference/cropland.md)                       | Cropland density for the world from three sources                               |
| [`elevation_3s`](https://rspatial.github.io/geodata/reference/elevation.md)                  | Elevation data for tile (3 seconds resolution)                                  |
| [`elevation_30s`](https://rspatial.github.io/geodata/reference/elevation.md)                 | Elevation data for by country (30 seconds resolution)                           |
| [`elevation_global`](https://rspatial.github.io/geodata/reference/elevation.md)              | Global elevation data (various resolutions)                                     |
| [`gadm`](https://rspatial.github.io/geodata/reference/gadm.md)                               | Administrative boundaries for any country in the world                          |
| [`world`](https://rspatial.github.io/geodata/reference/world.md)                             | Boundaries for the countries in the world                                       |
| [`landcover`](https://rspatial.github.io/geodata/reference/landcover.md)                     | Global landcover data                                                           |
| [`footprint`](https://rspatial.github.io/geodata/reference/footprint.md)                     | Human footprint data                                                            |
| [`osm`](https://rspatial.github.io/geodata/reference/osm.md)                                 | OpenStreetMap data by country                                                   |
| [`population`](https://rspatial.github.io/geodata/reference/population.md)                   | Download population density data                                                |
| [`soil_af`](https://rspatial.github.io/geodata/reference/soil_af.md)                         | Chemical and physical soil properties data for Africa for different soil depths |
| [`soil_af_water`](https://rspatial.github.io/geodata/reference/soil_af_water.md)             | Physical soil properties for Africa for water balance computations              |
| [`soil_af_elements`](https://rspatial.github.io/geodata/reference/soil_af_elements.md)       | Soil element concentration data for Africa                                      |
| [`soil_af_isda`](https://rspatial.github.io/geodata/reference/soil_af_isda.md)               | Soil data for Africa derived from the iDSA data set                             |
| [`soil_world_vsi`](https://rspatial.github.io/geodata/reference/soil_grids_vsi.md)           | Virtually connect to the global soilgrids data                                  |
| [`soil_world`](https://rspatial.github.io/geodata/reference/soil_grids.md)                   | Global soils data                                                               |
| [`sp_occurrence`](https://rspatial.github.io/geodata/reference/sp_occurrence.md)             | Species occurrence data from the Global Biodiversity Information Facility       |
| [`travel_time`](https://rspatial.github.io/geodata/reference/travel_time.md)                 | Travel time to cities and ports                                                 |
| [`worldclim_global`](https://rspatial.github.io/geodata/reference/worldclim.md)              | Global climate data                                                             |
| [`worldclim_country`](https://rspatial.github.io/geodata/reference/worldclim.md)             | Climate data by country                                                         |
| [`worldclim_tile`](https://rspatial.github.io/geodata/reference/worldclim.md)                | Climate data by tile                                                            |
